# Car Towing

## How to run

  Node.js should be installed
  Mongodb should be installed and runned

  * npm i

  # Without server autoreloading

    * node dev/server
    * open http://localhost:3000/

  # With server autoreloading

    * npm install nodemon -g
    * nodemon dev/server
    * open http://localhost:3000/
